# :lipstick: pug-sass-boilerplate :lipstick:

### Install dependencies.

```bash
$ yarn
```

**Note**: If you can't use [yarn](https://github.com/yarnpkg/yarn) for some reason, try `npm install`.

## Run

### watch and build

```bash
$ yarn dev
```
